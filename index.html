This is an OSSSBox Server.

The Online Simple Storage Server for S3

The goal of this project is to deliver a generic server back-end that can be used for many software development projects.

It provides a package of serverless functions that can be easily and quickly provisioned to:

  • Netlify as Netlify Functions, or to
  • Amazon AWS with AWS Lambda.

This package of serverless functions provide an authenticated, multi-user REST API interface to high-level generic project and asset definitions, with back-end storage on:

  • Amazon's Simple Storage Service (S3) or
  • Vultr's "Object Storage" (which is S3-compatible object storage).

This is a work in progress.
